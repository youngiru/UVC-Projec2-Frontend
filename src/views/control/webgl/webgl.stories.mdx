# WebGL(Three.js) Version

**Three.js에 대한 소개 및 설명입니다.**  

먼저 `WebGL`에 대한 설명에 앞서 간략하게 `Three.js`에 대한 설명입니다.
1. Three.js는 웹에서 그래픽을 손쉽게 해석할 수 있도록 도와주는 도구입니다.
2. 실제로 코드 상에서 WebGL 기술로 선 하나를 그릴 때에도 수학 연산이 필요합니다.
3. WebGL이 샤프심이라고 하면 Three.js는 샤프라고 할 수 있습니다. 행위를 쉽게 할 수 있도록 도와주는 도구입니다.
4. Three.js는 기본적으로 그래픽 언어와 같이 Scene 기준으로 그래픽 렌더링하며 동작합니다.
5. 구성 요소로는 크게 `scene`, `renderer`, `control`, `camera`, `light` 정도가 있습니다.

    - scene는 각종 3D 리소스들이 최종적으로 모여지는 공간
    - renderer는 웹에서 Three.js가 그래픽을 표현하는 모듈
    - control는 웹에서 마우스나 키보드를 통해 3D 컨텐츠를 조작할 수 있는 기능
    - camera는 배치된 웹의 3D 공간에서 컨텐츠를 볼 수 있도록 설치하는 눈의 역할
    - light는 현실에도 빛이 없으면 어두컴컴하듯이 3D 공간에서도 빛이 없으면 볼 수 없기 때문에 필요한 조명의 역할

| 구성 |     설명      |
|----------|:-------------:|
| Scene |  공간 |
| Renderer |   렌더링 도구   |
| Control | 3D 컨트롤러 도구 |
| Camera | 3D 카메라 도구 |
| Light | 3D 조명 도구 |
